# Отчет по лабораторной работе №1

**Выполнил:** Абакунов Кирилл, группа K3340  
**Тема:** сетевое программирование на Python (UDP/TCP/HTTP).

## Что сделано

В рамках лабораторной реализованы 5 небольших сетевых программ на Python. Каждое задание — отдельная пара (или набор) файлов сервера/клиента, работающих на `localhost` и разных портах.

## Используемые технологии

- Python 3
- Стандартные модули: `socket`, `threading`, `math`, `urllib.parse`

## Краткое описание заданий и кода

### Задание 1 — UDP клиент/сервер (порт 9999)

- Сервер принимает UDP‑сообщение и отправляет ответ.
- Клиент отправляет тестовую строку и выводит ответ.
- Основная логика: `recvfrom(...)` и `sendto(...)`.

Файлы: `task1_server.py`, `task1_client.py`

### Задание 2 — TCP “математический” сервер (порт 9998)

- Клиент отправляет числа `a,b`.
- Сервер вычисляет гипотенузу по теореме Пифагора и возвращает результат.

Файлы: `task2_server.py`, `task2_client.py`

### Задание 3 — HTTP сервер со статической страницей (порт 8080)

- Сервер принимает HTTP‑запрос и отдает HTML из файла `index.html`.
- Если файл не найден — возвращает 404.

Файлы: `task3_server.py`, `index.html`

### Задание 4 — Многопользовательский чат (порт 9997)

- TCP‑сервер принимает несколько клиентов одновременно.
- На каждого клиента создается поток; сообщения рассылаются остальным клиентам.
- В коде используется общий список подключенных клиентов и блокировка для синхронизации.

Файлы: `task4_server.py`, `task4_client.py`

### Задание 5 — Мини веб‑сервер (GET/POST) “Журнал оценок” (порт 8000)

- `GET /` показывает HTML‑страницу со списком оценок и формой.
- `POST /` принимает значения из формы, сохраняет их в памяти процесса и делает редирект обратно на `/`.
- Данные хранятся в словаре `grades` внутри процесса сервера (без БД).

Файл: `task5_server.py`

## Запуск

Для каждого задания: сначала сервер, затем клиент (если предусмотрен), либо открытие страницы в браузере.

```bash
# Задание 1
python3 task1_server.py
python3 task1_client.py

# Задание 2
python3 task2_server.py
python3 task2_client.py

# Задание 3
python3 task3_server.py
# открыть http://localhost:8080/

# Задание 4
python3 task4_server.py
python3 task4_client.py

# Задание 5
python3 task5_server.py
# открыть http://localhost:8000/
```
