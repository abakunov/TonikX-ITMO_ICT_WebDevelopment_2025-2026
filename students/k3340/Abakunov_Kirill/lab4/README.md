# Лабораторная работа №4 — Vue.js Frontend

Клиентская часть системы управления гостиницей, реализованная на Vue.js 3 с использованием Vuetify 3.

## Технологии

- **Vue.js 3** — Composition API
- **Vuetify 3** — Material Design UI
- **Pinia** — State Management
- **Vue Router 4** — Маршрутизация
- **Axios** — HTTP клиент
- **Vite** — Сборщик

## Требования

- Node.js 18+
- npm 9+
- Запущенный бэкенд из lab3

## Быстрый старт

### 1. Установка зависимостей

```bash
cd lab4
npm install
```

### 2. Запуск бэкенда (lab3)

```bash
cd ../lab3
docker-compose up -d
```

Или без Docker:

```bash
cd ../lab3
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### 3. Запуск фронтенда

```bash
npm run dev
```

Приложение будет доступно по адресу: **http://localhost:5173**

## Функционал

### Авторизация
- Регистрация новых пользователей
- Вход по логину и паролю
- Редактирование профиля
- Смена пароля

### Управление номерами
- Просмотр всех номеров
- Фильтрация по типу и доступности
- Создание, редактирование, удаление номеров
- История гостей в номере

### Управление гостями
- Список всех гостей
- Заселение и выселение
- Фильтрация по городу и статусу
- Просмотр кто убирает номер
- Просмотр одновременно проживающих гостей

### Управление персоналом
- Список служащих
- Приём на работу и увольнение
- Фильтрация по статусу

### Расписания уборки
- Расписание по дням недели
- Привязка служащих к этажам

### Отчёты
- Квартальные отчёты о доходах
- Статистика по этажам и номерам
- Визуализация топ номеров

## Настройка

### Переменные окружения

Создайте файл `.env`:

```env
VITE_API_URL=http://localhost:8000
```

## Команды

| Команда | Описание |
|---------|----------|
| `npm install` | Установка зависимостей |
| `npm run dev` | Запуск dev сервера |
| `npm run build` | Сборка для продакшена |
| `npm run preview` | Предпросмотр сборки |

## Документация

Для просмотра документации:

```bash
pip install mkdocs mkdocs-material
mkdocs serve
```

Документация будет доступна по адресу: **http://localhost:8000**

## CORS

Бэкенд настроен на приём запросов с `http://localhost:5173`. 
Если нужно изменить порт, обновите `CORS_ALLOWED_ORIGINS` в `lab3/hotel_system/settings.py`.
