# Система управления гостиницей - Frontend

Клиентская часть системы управления гостиницей, разработанная на Vue.js 3 с использованием Vuetify 3.

*Последнее обновление: 27 января 2026*

## Обзор

Данное приложение представляет собой SPA (Single Page Application), которое взаимодействует с REST API бэкенда на Django REST Framework.

## Основные возможности

- **Авторизация и регистрация** — полноценная система аутентификации с использованием токенов
- **Управление номерами** — просмотр, создание, редактирование и удаление номеров гостиницы
- **Управление гостями** — заселение, выселение, просмотр истории проживания
- **Управление персоналом** — найм, увольнение, просмотр служащих
- **Расписания уборки** — управление графиком уборки по этажам и дням недели
- **Отчёты** — квартальные отчёты о доходах гостиницы

## Технологии

| Технология | Версия | Назначение |
|------------|--------|------------|
| Vue.js | 3.4 | Основной фреймворк |
| Vuetify | 3.4 | UI библиотека |
| Pinia | 2.1 | State management |
| Vue Router | 4.2 | Маршрутизация |
| Axios | 1.6 | HTTP клиент |
| Vite | 5.0 | Сборщик |

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```

## Связь с бэкендом

Приложение подключается к бэкенду из lab3, работающему на `http://localhost:8000`. 
Убедитесь, что бэкенд запущен перед использованием фронтенда.
